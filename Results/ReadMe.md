
## Модель эмбеддингов
- **BAAI/bge-m3** (HuggingFace)
- Используется для преобразования текстовых чанков в векторные представления.
- Выбран как один из лучших вариантов для работы с русским текстом.
- Достаточно тяжелая модель

## База знаний
- Папка: `knowledge_base`
- Содержит текстовые файлы `.txt` с выборкой статей из https://warhammer40k.fandom.com/ru/wiki/
- В файлах выполнена замена основных понятий по словарю соответствий `terms_map.txt`

## Индекс векторного поиска
- Построен с помощью **Chroma** (`persist_directory="chroma"`).
- **Количество чанков в индексе:** примерно 200–250.
- Каждый чанк имеет уникальный `chunk_id` и метаданные (`filename` и `source`).

## Время генерации индекса
- Полная генерация (загрузка файлов → разбиение на чанки → создание эмбеддингов → построение базы) занимает в среднем 50 сек.
- Первая загрузка очень долгая из-за большой модели эмбеддинга

## Примечания
- Для тестов поиска использовалась функция `similarity_search_with_score` с фильтром по файлу и исключением дублирующихся чанков.
- Максимальное количество результатов на запрос — 3 наиболее релевантных уникальных чанка.
- Каждый файл разбивается на чанки по 1000 символов с перекрытием 200 символов для сохранения контекста.
